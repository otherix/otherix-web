What makes it different

Created from an image URL

No template entity, no registry. The agent materializes the image into a per-pool cache on first use.

First-class live migration

Peer-to-peer between agents, with the control plane out of the data path.

Explicit cluster resources

Storage pools, networks, and firmwares are real resources - not abstractions over a cloud provider.

One stateful service

Embedded etcd is the only datastore. No Postgres, no Redis, no message queue to operate.
